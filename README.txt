# Cultural Recipes Website

Upload this folder to GitHub Pages.

Important:
- Replace the placeholder images in `images/` with real food photos.
- Each recipe page currently points to JPG image names, for example `images/dumplings.jpg`.
- You can either rename your real photos to match these names, or edit the image path in each recipe HTML file.

Recommended image filenames:
- dumplings.jpg
- braised-pork.jpg
- roast-duck.jpg
- braised-prawns.jpg
- sea-bass.jpg
- radish-cake.jpg
- bamboo-mushroom.jpg
- cola-wings.jpg
